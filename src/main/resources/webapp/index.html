Project Planner - Group AE

Welcome

Plan any project, one milestone at a time.

With this site, your details and your project will be kept secure. If you need to share the project with anyone, you will be able to do so.

Each project will contain a list of milestones that you are planning to carry out for any project, for example a coursework project. As a user, you will be able to define your own milestones by giving them a description, an expected due date. Once the milestone is complete, users will be able to enter the actual completion date of the milestone.

You will also be able to use this planner to view any incomplete milestones, edit milestones and remove them from a project
